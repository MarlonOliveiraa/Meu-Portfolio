# Portfolio — Marlon Oliveira

Este repositório contém o código-fonte do meu portfólio pessoal como desenvolvedor.  
O projeto foi desenvolvido com foco em performance, acessibilidade e design intencional.

## Visão Geral

O objetivo deste projeto é apresentar meus trabalhos, habilidades técnicas e identidade profissional de forma clara e moderna.

## Tecnologias Utilizadas

- Vite
- TypeScript
- React
- Tailwind CSS
- shadcn/ui

## Requisitos

Para rodar o projeto localmente, você precisa ter instalado:

- Node.js (versão LTS recomendada)
- npm ou yarn
